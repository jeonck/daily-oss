# My Context

> 이 문서는 수집된 오픈소스 프로젝트의 "행동 판정" 기준이다.
> 여기 없는 주제는 원칙적으로 "무관" 또는 "학습"으로 판정한다.

## 내 환경/현황

- IT 분야 전반에 관심이 있는 실무자/의사결정자 — 특정 스택에 고정되어 있지 않고,
  개발자·DevOps·AI·기획·프론트엔드·백엔드·시스템 관리자·인프라 전 영역을 폭넓게 살핀다.
- 새로 알려지고 있는(신규 공개, 스타 급증, 커뮤니티에서 화제) 오픈소스 프로젝트 중
  "실제로 써볼 가치가 있는 것"을 빠르게 발견하는 것이 목적.
- 직접 운영 중인 특정 프로덕션 스택은 없음 — 즉시조치는 "당장 사용/도입해볼 만큼 완성도와
  파급력이 높은 프로젝트"를 의미하며, 특정 인프라 장애 대응이 아니다.

## 관심 분야 (학습 판정 기준)

### 개발자 도구
- CLI, 에디터/IDE 확장, 빌드/테스트 도구, 언어/런타임, 개발 생산성 도구

### DevOps / 인프라 / 시스템 관리
- CI/CD, 컨테이너/오케스트레이션(Kubernetes 생태계), IaC, 모니터링/관측성,
  네트워킹, 클라우드 네이티브 인프라 도구

### AI / LLM
- LLM 프레임워크, AI 에이전트, RAG/벡터DB, 파인튜닝/추론 최적화, MCP 서버,
  AI 코딩 도구

### 프론트엔드 / 백엔드
- 웹 프레임워크, UI 컴포넌트/디자인 시스템, 상태관리, API 프레임워크,
  데이터베이스/ORM, 백엔드 아키텍처 도구

### 기획 / 협업
- 프로덕트/프로젝트 관리, 노코드-로우코드, 협업·문서화 도구,
  데이터 시각화/대시보드

### 명시적 제외 (관심 분야처럼 보여도 무관 처리)
- 단순 튜토리얼/블로그 홍보, 특정 회사 제품의 마케팅성 릴리즈 노트
- 스타 수·활동성이 낮아 검증되지 않은 프로젝트(수집 단계에서 최소 스타 기준으로 1차 필터링됨)
- 게임, 하드웨어 전용 프로젝트 등 IT 실무와 무관한 오픈소스

## 판정 시 우선순위
1. 완성도가 높고 화제성이 커서 지금 바로 써볼 가치가 있는 신규/급부상 오픈소스 → 즉시조치 후보
2. 유용하지만 당장 급하지 않거나 아직 초기 단계인 것 → 백로그 후보
3. 도입 대상은 아니나 트렌드/아키텍처 파악에 도움되는 것 → 학습 후보
4. 그 외 일반 뉴스, 무관 주제, 마케팅성 글, 명시적 제외 항목 → 무관
